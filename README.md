## 🐾 JNSJ 통합 반려동물 케어 플랫폼


## ✨ 서비스 소개

--------

> 반려동물 용품 쇼핑과 병원 예약을 한 곳에서 이용할 수 있는 반려생활 전용 전자상거래 서비스입니다. </br>
상품 구매, 병원 예약, 자유게시판과 같은 기본 커뮤니티 기능을 제공해
반려인들이 일상에서 필요한 기능들을 간편하게 이용할 수 있도록 돕는 것이 목표입니다.

## 🎯 프로젝트 개요

--------


- 반려동물 용품 시장 확대에 따라 쇼핑·병원·커뮤니티 경험이 여러 서비스에 흩어져 있음
- 전자상거래 흐름(상품 → 장바구니 → 주문 → 결제 → 배송) 기반 구축
- 병원 예약 기능 및 자유게시판 추가로 반려인의 편의성 강화

>  전자상거래 + 병원 예약 + 커뮤니티 기능이 연동된 서비스 구축

## 🚀 주요 기능

------

### 🛍 상품

- 상품 CRUD
- 상품 활성화/비활성화
- 품절 처리
- 상품 재고 관리

### 🧺 장바구니

- 장바구니에 상품 추가
- 장바구니에 상품 조회 및 수량 변경, 삭제

### 📝 주문/결제

- 주문 CRUD
- 주문 상태 변경
- 결제 요청
- 결제 내역 조회 및 승인
- 결제 취소
- 전체 결제 내역 조회 
- 결제 상태 변경 

### 🚚 배송
- 배송지 CRUD
- 기본 배송지 설정
- 배송 상태 조회
- 배송 추적 조회
- 송장번호 등록
- 배송 상태 변경
- 택배사 CRUD
- 알림 발송

### 💬 리뷰
- 상품 리뷰 CRUD

### 👤 회원/관리자
- 회원가입
- 로그인/로그아웃
- 내 정보 CRUD
- 비말번호 변경
- 관리자 CRUD
- 회원 탈퇴 및 비활성화 

### 🐶 반려동물
- 반려동물 프로필 CRUD
